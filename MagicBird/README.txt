------------------------------------------------------------------------------------------------------------
Real-time Infoveillance of Twitter Health Messages (RITHM): Streamer 
------------------------------------------------------------------------------------------------------------

Command Line Args 
	-k keywords.txt (uses the keywords file to parse through the .json files)
	-e 				(emojifies the tweets)
	-c timeframe    (combines all the csv files together)
	-h				(help readout will be printed)

	
-k 
	uses the regular decoder to search through the .json files and pull out the right tweets
	
-e
	goes into the csv files and emojifies the tweets
	
-c 
	combines the csv files in either weeks, months, or all
	
-h 
	displays a help screen with the workings of the program
